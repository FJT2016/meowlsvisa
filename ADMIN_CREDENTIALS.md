## Admin Portal Access

1. Go to: https://evisa-meowls.preview.emergentagent.com/login
2. Enter your email and password (admin123)
3. Click "Sign In"
4. You'll be redirected to the dashboard
5. Click on "Admin" in the navigation bar to access the admin panel

## Admin Capabilities

✅ View all visa applications
✅ Review application details
✅ Approve applications (automatically sends email with AI-generated visa PDF)
✅ Reject applications (automatically sends kind rejection email)
✅ Add admin notes to applications
✅ Filter and search applications
✅ View statistics (Total, Pending, Approved, Rejected)

## Email Notifications

When an admin approves or rejects an application:
- **Approval**: Applicant receives email with AI-generated visa document PDF
- **Rejection**: Applicant receives professional rejection email with admin notes
